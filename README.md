# Quant Terminal v4

Bloomberg-style trading terminal with 4 engines (Mid-Point, 11-Trend, Gann 144, Smart Money Footprint), Alpha Scanner, news with world map, and interactive TradingView charts.

## Deploy

This Space uses Docker. Pushing to the `main` branch triggers auto-build.
